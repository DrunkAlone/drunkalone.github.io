1.三数之和

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

知识点：双指针。

本题采用排序+双指针的解法。首先对数组进行排序，随后遍历数组中的每一个元素，假定某次循环中元素为i，令左指针left = i + 1，右指针right指向数组的最后一个元素。

循环过程中有以下三种情况：

（1）当三个元素的和正好为0时，将该数对加入结果list，并将左指针右移一格，右指针左移一格。注意题干要求不能重复，所以该过程中需要判断移动后的元素是否和移动前的元素相等，如果相等则继续移动。

（2）三个元素的和小于0，说明左数太小，将左指针右移一格。

（3）三个元素的和大于0，说明右数太大，将右指针左移一格。
