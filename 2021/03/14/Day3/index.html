二、nowcoder客观题

  • 函数方法

    • 重载是同名不同参，参数列表必不同，返回值无关。如int A ( int a, int b ) 和double A ( int a, int b ) 并不是重载，但和int A ( double a, double b ) 是重载。

  • ORM对象关系映射

    一般ORM模型之间有以下对应关系：

    （1）表对应类；

    （2）记录对应对象；

    （3）表的字段对应类的属性。

  • 数据类型

    • java中文字符所占字节数取决于字符的编码方式：

      如果使用ISO8859-1编码，一个中文占1个字节；

      如果使用GBK编码，一个中文占2个字节；

      如果使用UTF-8编码，一个中文占3个字节。

    • java基本数据类型各占字节数为：

      byte：1个字节；

      short、char：2个字节；

      float、int：4个字节；

      double、long：8个字节。

  • JSP

    JSP共有以下六种基本动作：

    （1）include：在页面被请求时引入一个文件；

    （2）useBean：寻找或实例化一个JavaBean；

    （3）setProperty：设置JavaBean的属性；

    （4）getProperty：输出JavaBean的属性；

    （5）forward：把请求转到一个新的页面；

    （6）plugin：根据浏览器类型为Java插件生成OBJECT或EMBED标记。

  • Java基本类

    • Object类有以下方法：

      （1）clone()：创建并返回该对象的副本 ;

      （2）equals() ：判断相等；

      （3）finalize() ：垃圾回收器；

      （4） getClass() ：返回运行类；

      （5）notify() ：唤醒在此对象监视器上等待的单个线程；

      （6）notifyAll()：唤醒在此对象监视器上等待的所有线程；

      （7）hashCode()：返回该对象的哈希码；

      （8）toString() ：返回字符串；

      （9）wait()：等待线程。

  • 多线程

    • 集合中线程安全的类有：Vector、Stack、HashTable和Enumeration，此外均非线程安全。
    • yield（）方法可以使具有相同优先级的线程获得处理器，让当前正在运行的线程回到可运行状态。
    • Thread的sleep（）方法不会释放锁，而wait（）方法会释放锁。

  • Java定义

    Java中的关键字有abstract、assert、boolean、break、byte、case、catch、char、class、continue、default、do、double、else、enum、extends、final、finally、float、for、if、implements、import、int、interface、instanceof、long、native、new、package、private、protected、public、return、short、static、strictfp、super、switch、synchronized、this、throw、throws、transient、try、void、volatile、while、goto、const。

    而true、false、null属于特殊直接量。
